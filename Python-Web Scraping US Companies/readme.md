# 🐾 Colorado Homeless Pets — Data Wrangling Project

## 📌 Overview

This project focuses on cleaning, transforming, and analyzing multi-year Colorado animal shelter data to uncover trends in intake, outcomes, and community impact. The goal is to support programs and services that help every healthy and treatable pet find a safe outcome.

## 📊 Key Contributions

* Cleaned and transformed multi-year shelter datasets to enable accurate analysis
* Analyzed trends in:

  * Animal intake
  * Shelter outcomes
  * Community impact metrics

## 🛠️ Data Processing & Techniques

* Used **Python** and **Pandas** for data wrangling:

  * Reshaped datasets using `melt`
  * Split and merged fields
  * Standardized inconsistent formats for reporting
* Improved data quality and usability for downstream analysis

## 🌍 Data Enrichment

* Enhanced records with geographic insights by mapping ZIP codes
* Integrated external data using the **Google Maps API**

## 🤝 Collaboration

* Worked with a distributed team of volunteers across US and EU time zones
* Coordinated asynchronously to deliver analysis and insights

## 🎯 Impact

The processed data and insights support decision-making for animal welfare initiatives, helping improve outcomes for homeless pets across Colorado.
